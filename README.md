# 贪吃蛇游戏

一个使用HTML5 Canvas和JavaScript实现的经典贪吃蛇游戏。

## 游戏特点

- 简洁的界面设计
- 流畅的游戏体验
- 实时分数显示
- 支持键盘控制

## 如何游玩

1. 使用方向键（↑↓←→）控制蛇的移动方向
2. 吃到食物可以增加分数和蛇的长度
3. 撞到墙壁或自己的身体会导致游戏结束
4. 点击"重新开始"按钮可以开始新的游戏

## 在线游玩

## 本地运行

1. 克隆此仓库
2. 在浏览器中打开`index.html`文件
3. 开始游戏！

## 技术栈

- HTML5 Canvas
- JavaScript
- CSS3
